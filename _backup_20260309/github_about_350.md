# 1. 纯英文 About (适合 GitHub 官方简介栏，极简逼格)
WebViewHub is a native Windows desktop collaborative hub built with WPF & WebView2. It seamlessly integrates multiple AI web UI (ChatGPT, Gemini, etc.) into one dynamic dashboard layout. Features universal prompt broadcasting, cross-model context fetching, and a smart role-tag mechanism without using any APIs.

# 2. 纯中文 About (适合国内开源社区)
WebViewHub 是基于 WPF+WebView2 构建的原生 Windows 多窗口 AI 聚合工作台。它通过底层 DOM 注入和 DevTool 协议接管各大 AI 模型原生网页版，实现一屏多开、动态布局、跨端上下文逆向抓取以及一键弹窗群发提示词功能，是您的桌面级无 API 限制个人大模型协同中心。
