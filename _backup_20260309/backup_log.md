# WebViewHub 备份清单 (2026-03-09)

此目录包含了从项目根目录清理出的所有中间产物、构建日志和临时文件。

## 备份文件列表

| 原始路径 | 文件类型 | 说明 |
|---|---|---|
| `bin/` | 目录 | 编译输出的可执行文件及依赖 |
| `obj/` | 目录 | 编译过程中的中间对象文件 |
| `publish*/` | 目录 | 各种发布版本的输出目录 |
| `installer/` | 目录 | 安装包构建相关文件 |
| `*.log` | 文件 | 各种编译、构建错误日志 |
| `build*.txt` | 文件 | 构建过程记录 |
| `error*.txt` | 文件 | 错误信息记录 |
| `*.ps1` | 文件 | 各种修复和安装脚本 |
| `*.zip` | 文件 | 历史发布压缩包 |
| `Fixing WebView Sending Issues.md` | 文档 | 历史调试文档 |
| `history_*.md` | 文档 | 历史记录文档 |
| `github_*.md` | 文档 | GitHub 相关临时描述 |
| `turndown.js` | 脚本 | 被标记为可清理（项目内已有或为副本） |
| `Invoke-WebRequest/` 等 | 目录 | 异常生成的命令行残留目录 |

## 恢复指南
如果项目构建出现问题，可以将对应目录/文件移回原位。
大多数文件（日志、obj、bin）在下次执行 `dotnet build` 时会自动重新生成。
