**正文：**
天下苦来回切各种 AI 网页久矣！
我自己用 WPF + WebView2 搓了一个极其丝滑的**多模态 AI 协同终极桌面** —— **WebViewHub**。

它**不是**调用昂贵的 API，而是直接通过底层 DOM 注入自动化操作各大原生 AI 网站（完美绕开高防和风控）。
支持 ChatGPT、Gemini、豆包、Kimi 等网页**同屏无缝多开**。

**核心杀手锏：**
✨ **同屏多阵列排版：** 自由磁吸、瀑布流、双边阵列展示，支持随时切换手机版 UA 视图。
⚡ **一键中央群发：** 悬浮总控黑科技，打一句 Prompt，所有窗体自动敲字发送！
🔄 **跨模型接龙：** 指令支持打 `@角色提取` 功能（比如：让 ChatGPT 生成框架，再用 `@GPT` 提取给 Kimi 顺写正文）。
🛡️ **原生会话保持：** 各种 AI 免密免配置缓存，SQLite 记录所有布局参数。
